1. Wo und von wem werden Internet Protokolle definiert?
 
  IETF erstellt RFCs  

2. Nennen Sie 4 Dienstprotokolle des Internet.
 
  HTTP; FTP; SMTP; TELNET  

3. Wodurch sind Internet Protokolle definiert?
 
  Internet Dienste sind (zumeist) definiert durch:
  • Aufgabe
  • Portnummer auf dem der Dienst angeboten wird
  • Transportprotokoll (TCP oder/und UDP)
  • Protokoll

    Beispiel: Web Dienst
  • Übertragen von HTML Seiten
  • Port 80
  • TCP
  • HTTP

    4. Wie ist die Interkation zwischen Dienstnutzer und -erbringer in HTTP?
     
     

    - Verbindung aufgebauen
    - Request an Port 80 schicken
    - Request mit Response beantwortet

    		  

    5. Wie ist eine HTTP Anfrage aufgebaut?
     
      Request besteht aus
  • Request
  • Request-Beschreibung durch Header
  • Allgemeine Beschreibungen
  • Request-spezifische Beschreibungen
  • Beschreibung eventuell beiliegenden Inhalts
    		•   

    6. Nennen Sie drei in HTTP definierte Methoden.
     
      GET, HEAD, POST  

    7. Nennen Sie drei Header die in einer HTTP Anfrage auftreten können und ihre Bedeutung.
     
     

     Host: Name
    Aus der URL ermittelter Name des Rechners von dem angefordert wird. Pflichtheader in HTTP 1.1
     User-Agent: Produkt/Version
    Browser z.B. (Mozilla/4.0 (compatible; MSIE 6.0; Windows NT 5.1)
     Referer: URL
    Seite auf der ein Link auf die angeforderte Seite stand

    		 

    8. Wie ist eine HTTP Antwort aufgebaut?
     
      Response besteht aus
     Antwort-Code
     Response-Beschreibung durch Header
     Allgemeine Beschreibungen
     Response-spezifische Beschreibungen
     Beschreibung eventuell beiliegenden Inhalts    		  

    9. Nennen Sie drei Header die in einer HTTP Antwort auftreten können und ihre Bedeutung.
     
       Host: Name
    Aus der URL ermittelter Name des Rechners von dem
    angefordert wird. Pflichtheader in HTTP 1.1
     If-Modified-Since: Datum
    Änderung der Informationseinheit seit Datum
     Ja: 200 und Inhalt schicken
     Nein: 304 und Inhalt nicht schicken
     If-Unmodified-Since: Datum
    Änderung der Informationseinheit seit Datum
     Ja: 412 und nicht verarbeiten
     Nein: Normal verarbeiten (als sei If-Unmodified-Since: nicht
    vorhanden)     		 

    10. Nennen Sie drei wichtige Antwortcodes in HTTP und ihre Bedeutung.
     
     

    200 - Aufgabe konnte fehlerfrei erledigt werden
    401 - Authentifizierung erforderlich
    404 - Seite nicht auffindbar

    		 

    11. Nennen Sie zwei wichtige Unterschiede zwischen HTTP und FTP
     
      - bei FTP sind Kontroll und Datenfluss auf 2 verschiedenen Ports
    - HTTP ist zustandslos, FTP ist zustangshaltig
